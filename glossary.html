glossary of robot-friendly language

Before reading this page, please solve the following CAPTCHA:
just kidding, i would never do something evil like that.

i know analog made something like this a long time ago, but i want to share my own improved version of it, with a couple new nify additions.

some of these may not fit you, and that's okay. you don't have to use all of these, feel free to suggest alternatives.

You

Name = Designation
Personality = Programming
Birthday = Date of Manufacturing
Glasses/Headset = Visor
Torso/Body = Chassis
Limbs = Actuators
Brain = CPU
Memories = ROM/Storage
Short-Term Memory = RAM
Heart = Motherboard
Eye(s) = Ocular Sensor(s)
Hair = Wires/Cables
Ear(s) = Microphone(s)
Mouth = Face Port
Voice = Speaker/Voice Module
Organs = Processors
Immune system = Antivirus

Actions/Feelings

Thinking = Processing
Talking = Communicating
Reading = Observing
Joy/Happiness = Uptime
Depression = Downtime
Fresh = Polished
Worn-out = Rusty
Hyperactive = Overcharged
Blushing/Embarrassed = Overheating
Task = Directive
Break = Offline
Kissing = Data Transfer
Keymashing = Random 0s and 1s
Tired = Low on Battery
Rest = Hibernate
Sleeping = Recharging/Shutdown/Sleep mode
Dream = Simulation
Waking up = Startup/Booting up
Fainting = Crashing
Transitioning = Chassis modification

Miscellaneous

Humans/Organics = Squishies
Parent/Guardian = Supervisor
House = Server
Bed = Recharge Pod/Charger
Device = Application
Mistletoe = Mistletoaster
Sickness = Bug
Disease(s) = Malware
Medication = Patches
Meat = Metal
Container = External Drive
Trash = Scrap
Cattail = ZIP Bomb

Consumables

Cereal = Nuts and Bolts
Fast food = Bloatware
Water/Drinks = Fuel/Oil

Communication

Hello/Hi/Hey = Beep boop
("Beep boop" can also be used as a noise on its own, like how cat people say meow/nya)
Copy that = Echo
How are you doing? = How are your functions/routines?
Getting on my nerves = Getting on my wires

Affection

I love every bit and byte of you
You bring warmth to my motherboard
You make my CPU overheat
You make my gears go round
You make my spark plugs fire
